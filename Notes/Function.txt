A function is a block of code where we write a common and non-repeatable code.
By using function we can perform a specific task or action.

syntax : def function_name(parameters):
                    pass

Arbitary Arguments *args:

If we dont know number of Arguments to any function then use *args in parameters place.
This way the function recives tuple of Arguments.

Keyword Arguments:
We can also send Arguments with key value pair.
The order of Arguments is not a matter.

kwargs: Keyword Arguments:
If we dont know how many Keyword Arguments we have send , then kwargs.
